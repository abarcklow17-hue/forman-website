# **App Name**: Forman Haulage Pro

## Core Features:

- AI Load Vision Estimator: An AI-powered vision tool that allows users to upload photos of their junk pile to receive an immediate approximate volume and price estimation.
- Precision Booking System: A multi-step, conversion-focused interface for scheduling pickups with inline validation and date selection.
- SEO-Optimized Regional Hubs: Dynamically generated city-specific landing pages for Greeley, Fort Collins, and surrounding areas to drive organic search growth.
- Supabase Lead Vault: A central database architecture to store job details, customer metadata, and photo uploads for historical tracking.
- Transformation Showcase: A managed before/after gallery layout with lightbox capabilities to demonstrate quality of work and build trust.
- Unified Admin Command Center: A secure internal dashboard for managing incoming leads, updating availability, and reviewing image uploads from the field.
- Interactive Smart Concierge: A conversational decision-tree assistant that guides users through service tiers without complex external LLM requirements.

## Style Guidelines:

- A dark 'Nocturnal Industrial' theme based on #0B0B0B background, featuring a primary Rugged Orange (#F97316) for high-impact CTAs.
- Secondary UI surfaces use deep charcoal (#1A1A1A) with borders in a refined stone grey (#2A2A2A) to ensure structure over shimmer.
- Success accents use a vibrant Signal Green (#22C55E) and neutral body text in muted soft-whites (#D4D4D4) for premium readability.
- A refined font pairing featuring 'Space Grotesk' (sans-serif) for high-impact headlines to project tech-enabled efficiency, and 'Inter' (sans-serif) for clear, readable body text.
- Outline-style iconography from Lucide, utilizing thin stroke weights and consistent 2px width to match the rugged industrial aesthetic.
- Modular bento-box layouts and asymmetric containers that favor visual hierarchy and professional whitespace over repetitive centered text.
- Low-impact Framer Motion transitions; primarily utilizing 'gentle slide' entrance animations (15px Y-offset) and 200ms ease-in-out hover scales on cards.
